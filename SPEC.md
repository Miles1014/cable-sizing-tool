## 1. 系统定位与目标

### 1.1 定位
通用低压（≤1kV AC / ≤1.5kV DC）电缆选型计算工具，面向电气工程师日常设计使用。

### 1.2 MVP 范围
- **核心实现**: 载流量计算（含修正系数）
- **框架搭建**: 电压降校验、短路校验、保护协调（仅界面骨架+接口定义，逻辑后续填充）

### 1.3 设计原则
- 单 HTML 文件可运行，无外部依赖（CDN 引入仅限 UI 库）
- 纯中文界面
- 计算逻辑与 UI 严格分离（便于测试和维护）
- 数据表与计算引擎分离（便于更新和扩展）

## 6. 修正系数

### 6.1 环境温度修正系数 Ct

适用于空气中敷设（方法B1/B2/C/E/F/G）:

```javascript
/**
 * 温度修正系数
 * 索引: TEMP_CORRECTION[绝缘类型][环境温度] = 修正系数
 * 参考温度: PVC=30°C, XLPE/EPR=30°C (空气中)
```

## 8. 计算公式（载流量模块详细）

### 8.1 核心选型逻辑

```
目标: 找到满足 I'Z ≥ IB 的最小标准截面

其中:
  IB = 设计电流 (由负载计算得出)
  I'Z = IZ_table × Ct × Cg × Cs × Ch × Ccustom
  
  IZ_table = 查AMPACITY_TABLES得到的基准载流量
  Ct = 环境温度修正系数
  Cg = 群敷修正系数
  Cs = 土壤热阻修正系数 (仅方法D)
  Ch = 谐波修正系数 (可选)
  Ccustom = 用户自定义附加系数

算法:
  for section in STANDARD_SECTIONS (从小到大):
    IZ_base = lookup(method, insulation, conductor, cores, section)
    IZ_corrected = IZ_base × Ct × Cg × Cs × ...
    if IZ_corrected ≥ IB:
      return section  // 第一个满足条件的即为推荐截面
```

### 8.2 温度修正系数公式法

当环境温度不在查表值中时，用公式计算：

```
空气中 (B1/B2/C/E/F/G):
  Ct = sqrt((θmax - θa) / (θmax - 30))
  
  PVC:  Ct = sqrt((70 - θa) / 40)
  XLPE: Ct = sqrt((90 - θa) / 60)
  EPR:  Ct = sqrt((90 - θa) / 60)

直埋 (D):
  Ct = sqrt((θmax - θg) / (θmax - 20))
  
  PVC:  Ct = sqrt((70 - θg) / 50)
  XLPE: Ct = sqrt((90 - θg) / 70)

其中:
  θmax = 导体最高工作温度 (°C)
  θa = 实际环境温度 (°C)
  θg = 实际土壤温度 (°C)
```

### 8.3 设计电流计算（辅助功能）

```
三相系统:
  IB = P / (√3 × U × cosφ × η)

单相系统:
  IB = P / (U × cosφ × η)

其中:
  P = 有功功率 (W)
  U = 线电压 (三相) 或 相电压 (单相) (V)
  cosφ = 功率因数
  η = 效率 (电动机时考虑, 默认1.0)
```

### 8.4 插值方法

当环境温度或土壤热阻落在查表值之间时，使用线性插值：

```javascript
/**
 * 线性插值
 * @param x - 目标值
 * @param table - {key: value} 查找表
 * @returns 插值后的结果
 */
function interpolate(x, table) {
  const keys = Object.keys(table).map(Number).sort((a, b) => a - b);
  
  // 精确匹配
  if (table[x] !== undefined) return table[x];
  
  // 超出范围：外推（加警告）
  if (x < keys[0]) return table[keys[0]]; // 保守取边界值
  if (x > keys[keys.length - 1]) return table[keys[keys.length - 1]];
  
  // 找到相邻两点
  let lower, upper;
  for (let i = 0; i < keys.length - 1; i++) {
    if (keys[i] <= x && x <= keys[i + 1]) {
      lower = keys[i];
      upper = keys[i + 1];
      break;
    }
  }
  
  // 线性插值
  const ratio = (x - lower) / (upper - lower);
  return table[lower] + ratio * (table[upper] - table[lower]);
}
```

1. **载流量数据精度**: 数据表来源于 IEC 标准，实际工程中应以厂家提供的实测数据为准
2. **插值误差**: 非标准截面或非标准温度使用线性插值，有 ±2% 的误差
3. **方法 F 排列方式**: 实际排列方式（平行/三叶形/间距等）对载流量影响较大，当前简化处理
4. **谐波影响**: MVP 版本暂不实现谐波修正，后续 Phase 3 补充
5. **电缆并联**: 大电流回路的并联电缆均流问题需 Phase 3 处理
6. **热稳态假设**: 载流量基于热稳态条件，短时过载和循环负载需另行分析

## 14. 验证测试用例

### 测试 Case 1: 基础查表
- 输入: 方法E, XLPE, Cu, 三相, 设计电流 250A, 30°C, 单回路
- 预期: 推荐截面 70mm² (查表载流量 312A > 250A)

### 测试 Case 2: 温度修正
- 输入: 方法C, PVC, Cu, 三相, 设计电流 100A, 环境温度 45°C, 单回路
- 预期: 
  - Ct = 0.79 (PVC, 45°C)
  - 需要基准载流量 ≥ 100/0.79 = 126.6A
  - 查表: 35mm² = 134A > 126.6A → 推荐 35mm²

### 测试 Case 3: 群敷修正
- 输入: 方法B1, XLPE, Cu, 三相, 设计电流 80A, 30°C, 6回路成束
- 预期:
  - Cg = 0.57 (成束, 6回路)
  - 需要基准载流量 ≥ 80/0.57 = 140.4A
  - 查表: 50mm² = 175A > 140.4A → 推荐 50mm²

### 测试 Case 4: 直埋复合修正
- 输入: 方法D, XLPE, Cu, 三相, 设计电流 150A, 土壤温度30°C, 土壤热阻3.0, 3回路并排间距0.25m
- 预期:
  - Ct = 0.93 (XLPE, 直埋, 30°C)
  - Cs = 0.96 (热阻 3.0)
  - Cg = 0.70 (直埋, 3回路, 0.25m间距)
  - 综合系数 = 0.93 × 0.96 × 0.70 = 0.625
  - 需要基准载流量 ≥ 150/0.625 = 240A
  - 查表: 120mm² = 254A > 240A → 推荐 120mm²
